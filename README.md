# Ishuset Marselisborg Havn - Website

En moderne, responsiv hjemmeside for Ishuset Marselisborg Havn - en prisvindende isbutik i Aarhus.

## 🍦 Om Projektet

Dette er en moderne hjemmeside bygget med React, TypeScript, Tailwind CSS og Framer Motion. Websitet fremhæver kvaliteten, frisklavet is og den unikke atmosfære ved Marselisborg Havn.

## ✨ Funktioner

- **Immersive Hero Section** - Fuldt skærm landing med live åbningstider
- **Vores Historie** - Fremhæver frisklavet, økologisk og italiensk inspiration
- **Interaktivt Smagsmenu** - Filtrering mellem veganske og sæsonbetonede smage
- **Galleri** - Karusel med billeder og videoer
- **Særlige Services** - Festfryser leje og Isbevis gavekort
- **Kontakt & Kort** - Interaktivt kort og kontaktoplysninger

## 🛠️ Teknisk Stack

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS
- **Animationer**: Framer Motion
- **Ikoner**: Lucide React
- **Package Manager**: Bun

## 🎨 Design Principper

- **Minimalisme med bold typografi** - Rene layouts med stærk tekst
- **Mikro-animationer** - Smooth transitions og hover effekter
- **Responsivt design** - Mobile-first tilgang
- **Dansk sprog** - Primært dansk med mulighed for engelsk
- **Tilgængelighed** - Alt tekst og høj kontrast

## 🚀 Kom i Gang

### Forudsætninger

Sørg for at have Bun installeret:
```bash
curl -fsSL https://bun.sh/install | bash
```

### Installation

1. Installer dependencies:
```bash
bun install
```

2. Start udviklingsserveren:
```bash
bun start
```

3. Åbn [http://localhost:3000](http://localhost:3000) i browseren.
